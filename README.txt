== Description ==

WPTG Popup Comments allows you to view and comment any post without going to it's single page. With simple shortcode, you get a link to trigger the popup to appear displaying it's comments respectively.  

**Shortcode**
`
/**
 * post_id is the id of the post of which comments is displayed. 
 * class is the class attribute that you can add in the link
 * label is the the link label 
 **/
[wptg_popup_link post_id="1" class="comments_class" label="View Comments"]
`

== Installation ==

1. WPTG Popup Comments can be installed directly through your WordPress Plugins dashboard.
2. Click "Add New" and Search for "WPTG Popup Comments"
3. Install and Activate

Alternatively, you can also download the plugin using the download button on this page and then upload the wptg-popup-comments folder to the /wp-content/plugins/ directory then activate throught the Plugins dashboard in WordPress.


== Frequently Asked Questions ==

= Can we comment from the popup? =

Yes, you can comment from the popup. It works same as the default comments and the comment is submitted via AJAX.
